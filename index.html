Cursos

Todas tecnologias e ferramentas necessárias para desenvolver aplicações web.

  • - Define a estrutura do conteúdo da página.
  • - CSS (Cascading Style Sheets): Estiliza o conteúdo, controlando layout, cores e fontes.
  • - React: Biblioteca JavaScript para criar interfaces de usuário.
  • - Vue.js: Framework JavaScript progressivo para criar interfaces.
  • - Angular: Framework completo para desenvolvimento front-end.
  • - Svelte: Framework mais recente, focado em otimização e performance.
  • - Bootstrap: Framework CSS para design responsivo e componentes pré-construídos.
  • - Sass/Less: Extensões de CSS para facilitar a escrita e organização de estilos.
  • - TypeScript: Superset de JavaScript que adiciona tipagem estática.
  • - npm: Gerenciador de pacotes padrão para JavaScript.
  • - Yarn: Alternativa ao npm com foco em performance e segurança.

Todas tecnologias e ferramentas necessárias para desenvolver aplicações back-end.

  • - JavaScript (Node.js): Para aplicações server-side com alta performance.
  • - Python: Usado com frameworks como Django e Flask.
  • - Java: Com frameworks como Spring Boot.
  • - Ruby: Geralmente utilizado com Ruby on Rails.
  • - PHP: Popular com CMS como WordPress e frameworks como Laravel.
  • - C#: Usado em aplicações .NET.
  • - Go (Golang): Ideal para aplicações de alta performance.
  • - Rust: Focado em segurança e eficiência.
  • - Kotlin: Frequentemente usado em desenvolvimento server-side junto com Java.
  • - Express.js (Node.js): Minimalista para criar APIs e servidores.
  • - Django (Python): Framework completo com soluções prontas.

Todas tecnologias e ferramentas necessárias para o versionamento de código.

  • - Git: Sistema de controle de versão distribuído, no qual GitHub é baseado.
  • - Recursos Específicos do GitHub.
  • - Ferramenta de integração contínua (CI) e entrega contínua (CD).
  • - Permite automatizar workflows, como testes, builds e deploys.
  • - Baseada em YAML para configuração.
  • - Para hospedagem de sites estáticos diretamente a partir de um repositório GitHub.
  • - Suporta tecnologias como HTML, CSS, JavaScript, e geradores estáticos como Jekyll.
  • - Ferramenta de autocompletar baseada em inteligência artificial, integrada ao Visual Studio Code e outros editores.
  • - Repositório para armazenar pacotes como npm, Maven, NuGet, e Docker Images.
  • - GitHub pode ser usado com quase todas as linguagens e tecnologias.
